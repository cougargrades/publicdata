MUED 4242 -  Pedagogical Analysis of String Literature

Credit Hours: 2.0
Lecture Contact Hours: 2.0 Lab Contact Hours: 0.0
Prerequisite: MUSI 3215, Introduction to Large Forms.
Description
This course offers strategies and experience for analyzing string literature. The class is designed to help preservice string teachers better understand string literature and programming issues for public school groups of all levels.
Repeatability: No

Additional Fee: No
