CEA 3312 - Fundamentals of Computer Architecture

Credit Hours: 3
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: CEA 3338 ; all prerequisites must be completed with C- or higher.
Description
Quantitative basis for computer architecture, memory hierarchy design, processor design exploiting parallelism, pipelining, memory hierarchy design, trends in computing architectures.
Repeatability: No

Additional Fee: No
