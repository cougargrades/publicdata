ELET 4309 -  Object-Oriented Applicatn Prog

Credit Hours: 3
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: ELET 3425 or consent of instructor.
Description
Object-oriented concepts, implementation using modern programming languages such as C++, Java, and C#. Concepts of class, methods, overloading, abstract classes and plymorphism, inheritance, exception handling and files. Software design processes and tools, program efficiency, and differences and similarities between OOPLs.
Repeatability: No

Additional Fee: Yes
