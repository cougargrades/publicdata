COSC 1320 -  Object Oriented Programming

Credit Hours: 3.0
Lecture Contact Hours: 3.0 Lab Contact Hours: 0.0
Prerequisite: COSC 1304 OR ECE 3331 .
Description
Object oriented paradigm, UML modeling, GUI programming, and JAVA and C++ programming. Introduction to program specification, unit testing, and debugging.
