CEA 4321 -  Database Principles and Applications

Credit Hours: 3
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: CEA 3201 ; all prerequisites must be completed with C- or higher.
Corequisite: CEA 4339 .
Description
Introduction to databases, relational data modeling, SQL programming, conceptual modeling, database design theory and normalization, database security, data mining and data warehousing, distributed databases and NOSQL (not only Structured Query Language) systems.
Repeatability: No

Additional Fee: No
