CIS 6362 - Advanced SOC Analysis

Credit Hours: 3.00
Lecture Contact Hours: 3.0 Lab Contact Hours: 0.0
Prerequisite: CIS 6322.

Hands-on exploration of how to create a security operations center. Data harvesting, data analysis and display of pertinent security related data. Students will create a small SOC using ELK stack as part of the class.
