COSC 6315 - Data Science for Everyone

Credit Hours: 3
Lecture Contact Hours: 3 Lab Contact Hours: 0
Prerequisite: Graduate student; non-Computer Science major; MATH 5385.

This course is designed to teach to non computer science students how to conduct data analysis projects using commercially available tools. The course will cover the following topics: exploratory data analysis; data visualization; data preprocessing; unsupervised learning (clustering); supervised learning (classification and regression); documentation and presentation of data analysis results.
